# QA Task Automation

This project automates the **P2 Admin Dashboard** using **Python Selenium** with the **Page Object Model (POM)** design pattern.

## Features

- Login automation
- Navigate to Customers module
- Verify Overview dashboard
- Extract dashboard statistics
- Navigate to Customer Lists
- Capture first row customer data
- Validate data is not empty

## Tech Stack

- Python
- Selenium
- WebDriver Manager


## Run the project

Install dependencies: pip install -r requirements.txt


Run the test: python tests/test_customer_flow.py
